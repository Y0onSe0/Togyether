# 2. 화면 전환 흐름

```
[SCR-001 로그인]
    │ 성공
    ▼
[SCR-006 공지사항]  ← 로그인 후 진입점
    │ LNB → 상담
    ▼
[SCR-003 대기 중 상태] ◀─────────────────┐
    │ 상담 시작 버튼                       │ ACW 저장 완료
    ▼ (상태 전환, 화면 이동 X)             │
[SCR-003 상담 중 상태]                    │
    │ 통화 종료                           │
    ▼ (화면 이동)                         │
[SCR-004 ACW 카드 작성] ─────────────────┘

[SCR-003/004/005/006/007] → GNB 상담사명 클릭
    └─ 팝오버 표시
         └─ 로그아웃 → [SCR-001]

[SCR-003/004/005/006/007] → LNB
    ├─ 상담   → [SCR-003]
    ├─ 대시보드 → [SCR-005]
    ├─ 공지사항 → [SCR-006]
    └─ 상담 내역 → [SCR-007]

[SCR-001] → 계정 생성 링크 → [SCR-002] → 완료 후 [SCR-001]
```

### 4.3.4 AI 안내 카드 상태별 표시

| STEP 1 출력 상태 | AI 안내 카드 표시 내용 |
